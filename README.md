# Actividad2_senalesysistemas_fourier
Programa en Python que simula las señales en su naturaleza y bajo la transformada de Fourier.
SILVIA ESTEFANIA TREJO ROCHA | 91016 | IDS
En este programa podremos analizar y observar a los diferentes tipos de señales y
su comportamiento en el diferentes dominios, así como aplicando la Transformada de Fourier,
nos arrojará gráficas en las que esta información es visible. 
Se hizo uso de bibliotecas como numpy y matplotlib que fueron llamadas al inicio del programa.
